# SSTank

This is some kind of DDTank / Worms / Bad Eggs online game clone.
Written in C with SDL2. I want it to be portable to Windows and Linux at minimum.

The assets inside the `assets/` directory were grabbed from other games or the internet.
Since this is a hobby project and it will not be used for commercial purposes, I don't care too much about using stolen assets.
This is only to make the game look pretty while still in development.

The assets were mainly stolen from DDTank and Super Smash Flash (both Flash games). Others were just found in the internet or drawn by my girlfriend.

If you are curious, you can use `https://github.com/jindrapetrik/jpexs-decompiler` to dissect any Shockwave Flash game.
